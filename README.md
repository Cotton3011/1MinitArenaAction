# 1Minit Arena Action  
（Unity Prototype / ポートフォリオ）

## 🎮 概要  
UI・インゲーム・データ駆動設計を総合的に示すための、  
**1分で遊べる小規模アクションゲーム**です。

- TPS視点のシンプルなアクション  
- プレイヤー移動、攻撃、ダメージ処理  
- 敵AI（追跡・攻撃・死亡）  
- ScriptableObject によるデータ管理  
- UI（HPバー、ダメージ表示、リザルト）

Unity の基礎機能を使いつつ、  
実務でも評価される構造を意識しています。

---

## 🎯 目的  
本プロジェクトでは、以下のスキルを示すことを目的としています：

- **インゲームロジックの構築（中級）**  
- **UI設計・実装（上級寄り）**  
- **ステートマシンによる制御**  
- **NavMesh を使った簡易 AI**  
- **Hitbox / Hurtbox による攻撃判定システム**  
- **データ駆動型開発（ScriptableObject / JSON）**  
- **カメラ制御（Cinemachine）**

---

## 🛠 使用技術  
- Unity 2022.3 LTS  
- C#  
- CharacterController  
- NavMesh Agent  
- ScriptableObject  
- JSON 読み込み  
- Cinemachine  
- Unity UI (uGUI)
